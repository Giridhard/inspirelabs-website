# Inspirelabs corporate website

Static marketing site for **Inspirelabs Solutions Ltd** — the Hyderabad-based parent company behind GrabOn, GrabCash, GrabShare, RankDrive and Alternatives.co.

## Live demo

Deployed via GitHub Pages: https://giridhard.github.io/inspirelabs-website/

## Stack

Pure static HTML, CSS and a small amount of vanilla JavaScript. No framework, no build step. Fonts are Hanken Grotesk (display + body) and JetBrains Mono (labels and metadata), loaded from Google Fonts.

## Pages

- `index.html` — home
- `products.html` — five-platform portfolio
- `services.html` — four-pillar growth services
- `about.html` — company, founder note, three positions
- `careers.html` — hiring philosophy, criteria, benefits
- `contact.html`, `partnership.html`, `platform.html`, `ai-agents.html`, `insights.html`, `trust.html`, `privacy.html`, `terms.html`, `404.html`

## Local preview

```bash
python3 -m http.server 8000

```

## Brand

Built against the Inspirelabs Brand Guidelines V2 / 2026. Active orange `#FF7A45` is used sparingly for accent and CTA only.
